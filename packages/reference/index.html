Button / CTA

Tab to a button: the focus ring is styled off :focus-visible, not a class. Turn on OS High Contrast — the shape stays visible (the transparent border) and the ring maps to the system Highlight colour.

Accessible form field — DS-on vs DS-off

DS-ON — accessible default

We'll send your receipt here. Enter a valid email address.

Real <label for>, autocomplete, error tied via aria-describedby + aria-invalid, and the error reads with an icon + text — not colour alone.

DS-OFF — what ships without the system

Invalid

No label (placeholder vanishes on type; screen reader announces nothing), no autocomplete, error by red border + red word only — invisible under forced-colors and to colour-blind users.

Try the global-state a11y modes: zoom to 400% (layout reflows, no horizontal scroll — everything is in rem); enable OS reduced-motion (the button transition collapses to zero); enable OS forced-colors (the whole page remaps via the semantic tier).
